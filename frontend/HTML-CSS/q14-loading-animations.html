Loading Animations

Four CSS-only loading states — no JavaScript.

(A) Border Spinner

border + border-top-color + rotate(360deg)

One transparent border side creates the arc. Rotating the element 360° makes it appear to spin. animation: spin 0.8s linear infinite

(B) Bouncing Dots

translateY + animation-delay stagger

Same animation on all three dots. animation-delay: 0s, 0.15s, 0.3s creates the wave.

(C) Skeleton Screen

background-size: 200% + animating background-position

A gradient twice the element's width scrolls from right to left, creating the shimmer. Mimics real content layout.

(D) Progress Bar

width: 0% → 100% + ease timing-function

Animating width from 0% to 100%. The top bar uses ease (natural feel). Bottom bar uses linear (constant rate). Both loop with infinite.
